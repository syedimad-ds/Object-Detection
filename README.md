Classification Fluctuations (Flickering)Problem: Model kabhi-kabhi "Wrist Watch" ko "Cell Phone" ya "Bird" detect karta hai.Why? Model COCO dataset par trained hai jisme 'Wrist Watch' class nahi hai. Model nearest visual features ke base par "best fit" class (jaise phone) assign karne ki koshish karta hai.2. Accuracy vs Speed Trade-offProblem: Chote objects miss ho sakte hain.Why? Performance ke liye image ko 640x640 par resize kiya jata hai. Is scaling process mein fine spatial details loss ho jati hain.3. Hardware DependencyProblem: Alag-alag devices par FPS vary karta hai.Why? Ye WebGL backend par chalta hai. Bina dedicated GPU (jaise RTX 3050) wale devices par processing thodi slow ho sakti hai.🚀 How to Run LocallyIs repository ko clone karein:Bashgit clone https://github.com/syedimad-ds/Object-Detection.git
